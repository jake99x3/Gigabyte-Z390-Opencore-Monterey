# Gigabyte-Z390-Opencore- 

Gigabyte z390 I Aorus Pro Wifi Monterey 12.0.1 Opencore EFI

Sister repository to https://github.com/jake99x3/z390-I-Aorus-Pro-Wifi-Clover-Monterey
